Sistema de Gestión de Ventas
Descripción
Este proyecto consiste en un Sistema de Gestión de Ventas desarrollado como aplicación de escritorio.
Permite administrar clientes, productos, categorías y ventas, incluyendo el registro de facturas y consultas de compras realizadas por cada cliente.

El sistema está diseñado aplicando una arquitectura en capas (Entidad, Datos, Negocio y Presentación), buenas prácticas de programación y procedimientos almacenados en la base de datos
Tecnologías Utilizadas

Lenguaje: C#
Framework: .NET (Windows Forms)
Base de datos: SQL Server
Entorno de desarrollo: Visual Studio 2022

Uso o Ejecución

Para ejecutar el proyecto, se debe clonar el repositorio desde GitHub, abrir la solución en Visual Studio 2022, configurar correctamente la cadena de conexión a SQL Server, ejecutar previamente los scripts de creación de la base de datos y sus procedimientos almacenados, y finalmente ejecutar la aplicación desde el entorno de desarrollo.
